## 一、产品概述

### 1.1 产品定位

P6 AI企微侧边栏是嵌入企业微信客户端的统一AI入口，在客服会话场景中为一线业务人员提供即时的客户信息查看、AI话术推荐、营销辅助和快捷操作能力。产品跨角色覆盖管家、市场人员和咨询师三大业务群体，通过企微侧边栏这一高频触点，将AI能力无缝嵌入日常沟通工作流，实现"沟通即服务、对话即营销"。

### 1.2 产品背景

丽滋卡尔一线业务人员在与客户沟通时面临以下痛点：
- **信息割裂**：客户档案在CRM系统中，沟通时需反复切换系统查看
- **话术不统一**：不同人员回复质量参差不齐，缺乏标准化话术支持
- **营销低效**：客户分层和活动推荐依赖人工判断，精准度和时效性不足
- **操作繁琐**：预约、查报告等操作需要登录多个系统，效率低下
- **通话记录缺失**：电话沟通内容未被有效记录和利用

### 1.3 目标用户

| 角色 | 人数(估) | 核心使用场景 | 高频操作 |
|------|----------|-------------|----------|
| 管家 | 20-30人 | 客户日常沟通维护 | 查看客户画像、话术推荐、一键预约 |
| 市场人员 | 8-12人 | 营销活动推广 | 客户分层、活动推荐、素材生成 |
| 咨询师 | 10-15人 | 客户咨询解答 | 查报告、生成方案、话术推荐 |

### 1.4 核心价值主张

- **零切换体验**：在企微会话窗口内直接获取客户信息和AI辅助，无需切换系统
- **AI即时赋能**：实时话术推荐和营销建议，提升沟通质量和转化率
- **全角色覆盖**：一个入口适配三种角色，统一技术架构降低维护成本
- **营销智能化**：AI驱动的客户分层、活动匹配和素材生成，提升营销ROI

### 1.5 产品愿景

打造企业微信生态内的"AI业务助手"，让每一次客户沟通都有AI赋能，成为丽滋卡尔一线业务人员不可或缺的智能工具。

### 2.2 技术选型

| 层级 | 技术方案 | 说明 |
|------|----------|------|
| 前端 | Vue 3 + TypeScript | 企微侧边栏H5应用，轻量快速 |
| UI框架 | Vant 4 | 移动端UI组件库，适配企微环境 |
| 状态管理 | Pinia | 轻量级状态管理 |
| 后端 | Node.js + NestJS | 轻量级BFF（Backend for Frontend） |
| AI服务 | Python FastAPI | AI推理服务（话术生成/分层/素材） |
| 消息队列 | Redis Streams | 轻量级事件通知 |
| 缓存 | Redis + Caffeine | 客户画像缓存TTL 5分钟、会话上下文实时维护、话术推荐缓存10分钟。缓存策略详见优化方案09 |
| 数据库 | MySQL | 业务数据持久化 |

> **技术栈独立性说明**（详见优化方案08）：P6独立采用Vue 3 + Vant 4的原因是企微侧边栏H5应用的轻量化需求和企微生态适配。
> 应对策略：短期保持Vue + 抽取@lizkal/shared-*共享npm包（types/utils/api/constants），移除NestJS BFF层改为直连Spring Boot。
> 共享包可覆盖30-40%代码逻辑，Q3评估是否需迁移至React。

### 2.3 企微集成架构

```
企微客户端
    │
    ├── 企微JS-SDK ──→ 获取会话上下文（当前客户ExternalID）
    │
    ├── 侧边栏H5应用 ──→ BFF服务 ──→ CRM系统（客户画像）
    │                            ├──→ AI推理服务（话术/分层/素材）
    │                            ├──→ 预约系统（一键预约）
    │                            ├──→ 报告系统（查报告）
    │                            └──→ 方案系统（生成方案）
    │
    └── 企微回调 ──→ 会话归档回调 ──→ 通话记录处理
```

### 2.4 数据流转

1. **会话触发**：用户打开客户会话 → 企微JS-SDK获取ExternalUserID
2. **画像加载**：BFF → CRM系统查询客户画像 → Redis缓存 → 前端渲染
3. **话术推荐**：获取会话上下文 → AI服务生成推荐话术 → 前端展示
4. **操作执行**：用户触发快捷操作 → BFF调用对应系统API → 返回结果

## 四、5大功能模块详细设计

### 4.1 客户画像即时查看

**功能描述**：打开客户会话时，侧边栏自动加载并展示该客户的核心画像信息，无需切换到CRM系统。

**画像卡片展示内容**：

**基本信息区**：
- 客户头像 + 姓名 + 性别年龄
- VIP等级标签（普通/银卡/金卡/铂金）
- 客户标签云（如：高净值、体检常客、关注心血管等）
- 上次到店日期 + 上次沟通日期

**健康摘要区**：
- 最近一次体检日期和套餐
- 关键异常指标摘要（如：血压偏高、血糖异常等）
- 医生建议摘要（3条以内）
- 复查提醒（距下次复查天数）

**消费信息区**：
- 累计消费金额
- 最近消费记录（最近3条）
- 会员积分余额
- 未使用的优惠券/权益

**沟通历史区**：
- 最近5条沟通记录摘要（日期+渠道+主题）
- 上次沟通要点标注
- 待跟进事项提醒

**交互设计**：
- 默认展示"基本信息"和"健康摘要"，其他区域折叠
- 点击"查看完整画像"跳转CRM系统详情页
- 画像数据实时从CRM系统同步，缓存有效期5分钟
- 加载中展示骨架屏

### 4.2 AI话术推荐

**功能描述**：根据当前会话上下文和客户画像，AI实时推荐适合的回复话术。

**话术推荐类型**：

| 场景 | 触发条件 | 推荐内容示例 |
|------|----------|-------------|
| 欢迎话术 | 新客户首次沟通 | "XX先生/女士您好，我是您的专属管家..." |
| 咨询回复 | 客户询问套餐信息 | 套餐对比介绍+推荐理由 |
| 异常关怀 | 客户有体检异常指标 | "关注到您上次XX指标偏高，建议..." |
| 复查提醒 | 客户接近复查时间 | "距离您的复查还有X天，建议提前预约..." |
| 投诉安抚 | 检测到客户负面情绪 | 安抚话术+解决方案+升级路径 |
| 节日问候 | 节假日期间 | 节日祝福+专属优惠推荐 |
| 续费引导 | 会员即将到期 | 续费优惠+权益说明 |

**交互设计**：
- 话术推荐卡片显示2-3条推荐话术
- 每条话术旁边有"一键发送"按钮（点击后自动填充到企微输入框）
- 支持"换一批"刷新推荐
- 支持话术微调编辑后发送
- 用户可对话术点赞/点踩，用于模型优化
- 话术推荐响应时间≤3秒

### 4.3 AI营销助手

**功能描述**：为市场人员和管家提供智能营销辅助能力，包括客户分层、活动推荐和营销素材生成。

#### 4.3.1 客户分层

**分层维度**：
- **价值分层**：高价值/中价值/低价值/潜力客户
- **活跃分层**：活跃/沉默/流失/已流失
- **需求分层**：体检需求/健康管理/慢病管理/美容保健
- **生命周期**：新客/成长/成熟/衰退

**展示方式**：
- 当前客户的分层标签可视化展示
- 分层依据说明（AI给出分层理由）
- 同分层客户群体统计

#### 4.3.2 活动推荐

**功能详情**：
- 基于客户画像和分层，AI推荐最匹配的当前在线活动
- 展示活动名称、优惠内容、有效期、预计匹配度
- 一键生成个性化的活动推介话术
- 支持发送活动海报/链接到会话

#### 4.3.3 素材生成

**可生成素材类型**：
- **文案生成**：朋友圈文案、群发文案、个性化消息
- **海报文案**：活动推介海报上的文案内容
- **方案文档**：个性化健康管理方案摘要
- **邀请函文案**：活动邀请、体检邀请等

**交互流程**：
1. 选择素材类型
2. 选择目标客户（当前客户/批量客户）
3. 选择活动/主题
4. AI生成素材（支持多个版本）
5. 用户编辑调整
6. 一键发送/保存

### 4.4 快捷操作入口

**功能描述**：将高频操作以快捷按钮形式集成到侧边栏，一键完成跨系统操作。

**快捷操作列表**：

| 操作 | 功能说明 | 适用角色 |
|------|----------|---------|
| 一键预约 | 快速为当前客户创建预约 | 管家、咨询师 |
| 查报告 | 查看客户最新体检报告摘要 | 咨询师、管家 |
| 生成方案 | AI生成个性化健康管理方案 | 咨询师 |
| 发送提醒 | 快速发送预约/复查/缴费提醒 | 管家 |
| 创建工单 | 创建客户服务工单 | 全角色 |
| 添加标签 | 为客户添加/修改标签 | 全角色 |
| 查看排班 | 查看医生当日排班空位 | 管家 |
| 转介绍 | 生成转介绍专属链接 | 市场 |

**一键预约详细流程**：
1. 点击"一键预约"按钮
2. 系统自动填充客户信息
3. 展示可选套餐列表（AI推荐排序）
4. 选择预约日期和时段（标记空闲/忙碌）
5. 确认提交
6. 预约成功自动在会话中发送确认消息

**查报告详细流程**：
1. 点击"查报告"
2. 展示客户历史报告列表（最近5份）
3. 点击查看报告摘要（关键指标+异常标注+医生建议）
4. 支持将报告摘要以卡片形式分享到会话

### 4.5 通话总结

**功能描述**：对电话沟通进行智能录音转写和关键信息提取。

**核心功能**：
- **语音转文字**：通话录音自动转写为文本
- **智能摘要**：AI提取通话关键信息（客户诉求、承诺事项、待办跟进）
- **情绪分析**：分析通话中客户的情绪倾向（满意/中立/不满）
- **待办提取**：自动识别通话中的待办事项并生成跟进提醒
- **沟通记录**：通话总结自动写入客户沟通历史

**展示方式**：
- 通话结束后在侧边栏自动展示通话总结卡片
- 包含：通话时长、关键要点（3-5条）、情绪标签、待办列表
- 支持人工修正和补充
- 一键将通话摘要同步到CRM系统

## 五、跨角色适配方案

### 5.1 角色识别机制

通过企微登录账号关联后台角色信息，自动识别当前用户角色，展示对应的功能模块和内容。

### 5.2 各角色功能矩阵

| 功能模块 | 管家 | 市场人员 | 咨询师 |
|----------|------|----------|--------|
| 客户画像 | 全部信息 | 基本+消费 | 全部信息 |
| AI话术推荐 | 服务话术 | 营销话术 | 专业话术 |
| 客户分层 | 查看 | 全功能 | 查看 |
| 活动推荐 | 推荐+发送 | 全功能 | 推荐+发送 |
| 素材生成 | 基础文案 | 全类型 | 方案类 |
| 一键预约 | 可用 | 不可用 | 可用 |
| 查报告 | 可用 | 不可用 | 可用 |
| 生成方案 | 不可用 | 不可用 | 可用 |
| 通话总结 | 可用 | 可用 | 可用 |

### 5.3 角色切换

- 默认根据账号角色自动适配
- 部分用户可能有多角色权限，支持侧边栏标题栏切换角色
- 角色切换后，功能模块和内容实时更新

### 5.4 各角色界面差异

**管家视图**：
- 默认展开：客户画像 + 话术推荐
- 快捷操作突出：一键预约、发送提醒

**市场人员视图**：
- 默认展开：客户分层 + 营销助手
- 快捷操作突出：活动推荐、素材生成、转介绍
- 强调营销转化类功能

**咨询师视图**：
- 默认展开：客户画像（含健康摘要）+ 话术推荐
- 快捷操作突出：查报告、生成方案
- 强调专业咨询类功能

## 七、典型应用场景

### 场景一：管家日常客户沟通

**角色**：管家小王

**场景描述**：
VIP客户张先生通过企微咨询体检套餐事宜。

**操作流程**：
1. 小王打开与张先生的企微会话，侧边栏自动加载
2. **客户画像卡片**显示：金卡会员、上次体检3个月前、血压指标偏高、积分余额2800
3. 张先生发来消息："想了解一下今年的体检套餐"
4. **AI话术推荐**自动生成3条回复：
   - 话术A：介绍新年升级套餐，结合张先生的血压情况推荐增加心血管专项
   - 话术B：推荐VIP专属套餐，突出会员权益
   - 话术C：结合上次体检结果，建议关注的重点项目
5. 小王选择话术A，微调后一键发送
6. 张先生表示感兴趣，小王点击"**一键预约**"
7. 系统自动填充张先生信息，小王选择套餐和时间
8. 预约成功，自动在会话中发送预约确认卡片

**价值体现**：沟通效率提升60%，客户体验大幅提升

### 场景二：市场人员精准营销

**角色**：市场专员小李

**场景描述**：
公司推出春季体检优惠活动，小李需要精准触达目标客户。

**操作流程**：
1. 小李打开营销助手，查看**客户分层**
2. AI将客户分为高价值活跃、高价值沉默、中价值活跃、低价值等群组
3. 针对"高价值沉默"群体，AI推荐春季活动（匹配度92%）
4. 小李使用**素材生成**功能：
   - 选择"朋友圈文案"类型
   - 选择春季活动主题
   - AI生成3个版本的文案
5. 小李选择最佳版本，微调后发布到朋友圈
6. 同时选择"个性化消息"，AI为每位客户生成个性化推荐语
7. 批量发送个性化消息给目标客户群

**价值体现**：营销精准度提升40%，客户响应率提升25%

### 场景三：咨询师专业咨询

**角色**：咨询师刘老师

**场景描述**：
客户李女士来电咨询体检报告异常指标。

**操作流程**：
1. 通话过程中，侧边栏自动展示李女士的客户画像
2. 刘老师点击"**查报告**"，查看最近一次体检报告摘要
3. 报告显示：甲状腺结节（3类）、维生素D偏低、BMI偏高
4. **AI话术推荐**自动生成专业咨询话术：
   - 甲状腺结节3类的专业解释和随访建议
   - 维生素D补充方案建议
   - 体重管理建议
5. 通话结束后，**通话总结**自动生成：
   - 关键要点：客户咨询甲状腺结节，已解释3类风险较低
   - 承诺事项：3个月后复查甲状腺超声
   - 待办：为客户生成个性化健康管理方案
6. 刘老师点击"**生成方案**"，AI自动生成含复查计划的健康管理方案
7. 方案卡片通过企微发送给李女士

**价值体现**：咨询专业度提升，客户信任感增强，复购率提升
